# Homework-VueJS-JS

## Usage

Install

```sh
$ npm install
# --- OR ---
$ yarn
```

Development server

```sh
$ npm run start
# --- OR ---
$ yarn start
```

Build development

```sh
$ npm run build:dev
# --- OR ---
$ yarn build:dev
```

Build production

```sh
$ npm run build:prod
# --- OR ---
$ yarn build:prod
```
